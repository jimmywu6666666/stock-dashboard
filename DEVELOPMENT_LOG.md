# 开发维护日志

## 2026-06-17

本轮集中完成了股市信息综合看板从个人本地看板到可部署、多用户、移动端友好的在线系统的主要功能建设。

### 看板与数据

- 完成 PC 和手机端自适应看板，系统名称统一为“股市信息综合看板”。
- 核心行情增加上证指数、纳斯达克综合、国际现货金、布伦特原油、BTC/USDT、人民币汇率等数据。
- 增加 A 股大盘模块，展示成交额、较昨日放量、涨跌分布、涨跌家数，并优化盘前/未开盘时的数据展示。
- 金十重要事件、东方财富资讯热榜、东方财富热股、概念板块涨幅榜完成整合展示。
- 概念板块涨幅榜支持查看相关股票，相关股票可继续打开个股详情。
- 股吧热门帖子调整为当天不足 10 条时自动向前补足，并支持帖子详情、图片、回复展示。
- 近期公告加入个股详情页，支持近一周公告展示、公告正文弹层和评论读取。

### 自选股与账户

- 自选股支持添加、删除、排序、设置成本和持仓，并在删除、添加、修改成本/持仓时进行二次确认。
- 自选股卡片展示现价、涨跌幅、概念标签、市值、今日盈亏、今日盈亏比、总盈亏、总盈亏比。
- 增加自选股组合汇总，展示所有自选股的总市值、今日盈亏、今日盈亏比、总盈亏、总盈亏比。
- 上传截图识别自选股接入服务端 OCR，识别结果先进入确认弹层；已存在股票会比对成本和持仓，确认后再更新。
- 管理员支持创建账号、重置密码、设置用户名称、设置普通账号到期日，并记录最后活跃时间。
- 管理员可切换到指定用户的数据视角查看该用户自选股、股吧帖子等内容，但管理页不再直接展开用户自选股详情。
- 普通用户增加自行修改密码功能。
- 登录页去掉默认 admin 填充，避免正式环境暴露默认账号暗示。

### 个股详情与 AI 分析

- 增加个股查询入口，支持输入代码或名称模糊匹配待选股票。
- 个股详情页展示名称、代码、现价、今日涨跌和比例、概念、走势、资金流向、公告和股吧热门贴。
- 个股行情字段增加今开、最高、最低、成交额、换手率、总市值、流通市值、市盈率(动)等。
- 走势区域从静态图片改为 H5 图表控件，支持分时、日 K、周 K、月 K 切换，并显示开收高低、涨幅、涨跌、成交、振幅等信息。
- 接入 DSA AI 分析能力，并处理用户隔离，管理员切换其他用户时不会看到管理员自己的 AI 查询记录。
- 管理员增加“一键分析自选股”，仅分析管理员本人自选股，支持重复任务识别和可选强制重新分析。
- 优化 AI 分析历史列表滚动，避免自动刷新或点击结果时跳回顶部。

### 移动端体验

- 手机端 UI 按 App 化方向重做，增加启动界面，减少进入后大量模块同时等待刷新造成的空白感。
- 手机菜单重新分组：资讯、热度、自选股、使用手册等更贴近使用场景；个股查询移动到更靠前位置。
- 手机行情卡一行展示两个，资金流向改为更紧凑布局。
- 增加添加到桌面/加入 Dock 的快捷入口指引，覆盖 Windows、macOS、iPhone、安卓和鸿蒙。
- 锁定移动端页面缩放，减少双击、双指、输入框聚焦导致页面被误放大的问题。
- 修复多个模块自动刷新导致阅读、输入、弹层和滚动位置被打断的问题。

### 收盘日报

- 增加自选股每日收盘总结，不走 DSA 链路。
- 日报基于用户自己的自选股生成，包含组合汇总、个股盈亏、热门概念、近一周公告等精简内容。
- 支持用户自行配置邮箱收件地址、开启/关闭日报、发送测试日报。
- 定时任务默认每天 `Asia/Shanghai 16:30` 执行，只给未过期、已启用、有自选股的用户发送。
- 增加日报发送记录和管理员查看状态。
- 删除 Telegram 相关实现，当前只保留邮件发送渠道。
- 优化邮件 HTML 排版，去掉股吧热帖，仅保留公告摘要，适合手机邮件阅读。

### 部署与维护

- 完成 1Panel + Docker Compose 部署流程，支持域名和 SSL 访问。
- 增加部署文档、环境变量示例、服务器同步部署脚本和正式服务健康检查。
- 增加应用版本接口和前端代码更新提示，方便用户获取最新系统版本。
- 使用 SQLite 保存用户、自选股、配置、AI 任务、日报设置和发送记录。
- `.env`、`.env.deploy`、数据库和构建产物已加入忽略规则，避免密钥和运行数据进入 Git。
- 使用手册同步补充账号登录、自选股、截图识别、个股查询、AI 分析、收盘日报和添加到桌面的说明。
